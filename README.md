# Dom List

## Description
Une application web permettant de gérer une liste d'éléments en utilisant le DOM.

## Fonctionnalités
- Ajout d'éléments à la liste.
- Suppression d'éléments de la liste.
- Modification des éléments existants.

## Technologies utilisées
- HTML
- CSS
- JavaScript
